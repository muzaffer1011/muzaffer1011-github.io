Simply set up your profile using the edit profile page. Ensure you provide detailed information. Project & hiring managers will be able to view this information if you set your profile as visible.








2. Request Performance Feedbacks

Now that your profile is set up, you are ready to start invite your prior clients and colleagues for your performance feedbacks. Simply go to the request feedback page and start sending invites.
